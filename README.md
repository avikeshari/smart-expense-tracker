# Smart Expense Tracker

A React + Tailwind CSS expense tracker with:

- Expense entry form
- Category-wise totals
- Search, filters, and sorting
- Pie and bar charts
- localStorage persistence

## Tech Stack

- React JS
- Tailwind CSS
- localStorage
- Recharts

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
